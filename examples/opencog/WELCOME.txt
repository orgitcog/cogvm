QUICK START:

  1. Start CogServer:
     $ cogserver

  2. Connect to CogServer (in new terminal or via Claude AI):
     $ telnet localhost 17001
     
  3. Enter Scheme shell (once connected):
     opencog> scm

  4. Try basic Atomese:
     guile> (use-modules (opencog))
     guile> (ConceptNode "hello")

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

HELPFUL COMMANDS:

  ~/opencog-examples/opencog-start.sh    Interactive quick start
  guile -l ~/opencog-examples/FILE.scm   Run example script
  rlwrap telnet localhost 17001          Connect with history

CLAUDE AI INTEGRATION:

  Use the Claude AI panel (robot icon) to interact with OpenCog
  using natural language. Claude can:
  
  • Translate your requests into Atomese
  • Execute Scheme commands in CogServer
  • Query the AtomSpace
  • Explain OpenCog concepts
